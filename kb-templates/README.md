# kb-templates

Reusable document templates for the knowledge base. Copy one when starting a new pattern, decision record, snippet, runbook, comparison, idiom, or ecosystem guide. Each template carries frontmatter prompts and section scaffolding so both humans and AI fill them out consistently.

## Contents

- `pattern.md` — language-agnostic design pattern
- `adr.md` — architecture decision record
- `snippet.md` — copy-paste code with context
- `runbook.md` — operational procedure
- `comparison.md` — cross-language Rosetta Stone entry
- `idiom.md` — language-specific idiom
- `ecosystem-guide.md` — library/framework selection guide
- `openapi.md` — OpenAPI 3.1 starter
- `protobuf.md` — protobuf service starter

## When to pull into a workspace

When starting new knowledge base docs. Not needed for day-to-day project work.
